## とりあえずluigiのまとめ  

`requires`  
　依存関係にある別のTaskを呼ぶ。 
  ```
  Task B:
    requires(Task A)
  ``` 
  Aのrunの結果がBにかえる

`run`  
  実行内容   
  requiresのrunの結果をinputとして受け取る  
  結果をoutputにretuenする  

`output`  

  Taskの出力
